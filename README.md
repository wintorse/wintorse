## Works
<img src="https://user-images.githubusercontent.com/80797789/223946898-59aa360d-aec7-468e-9c43-b6118cd24097.jpg" width="80%" />

**AgriAR**(2021)
- AR pilot system that visualize the current leaf amount(LAI) of your greenhouse plants and existence of pests, and show a desirable plant sample immersively in 3D
- Microsoft HoloLens 2, Unity
- As my Senior thesis, I conducted needs analysis and implemented a prototype system.
- This research was orally presented at [Japan Geoscience Union Meeting 2022(日本地球惑星科学連合2022年大会)](https://www.jpgu.org/meeting_j2022/) and [Visws2022(可視化情報学会第5回ビジュアリゼーションワークショップ)](https://www.vsj.jp/visws2022/).

**[一葉(Ichiyo)](https://archive.iiiexhibition.com/log/i3e24#work4)**(2022)
- media artwork
- 2D code on leaves: laser engraved
- Web app: made with Nuxt.js and firebase
- Web UI demo: https://youtu.be/hZ9n2qEj1pA
- exhibited as a part of iii Exhibition 2022, with four members
- featured in [美術手帖](https://bijutsutecho.com/magazine/news/exhibition/26372)
- got an Honorable Mention at [GOOD DESIGN NEW HOPE AWARD 2023](https://newhope.g-mark.org/award/2023/23NHA040011.html)

**[近見知新聞解読依頼](https://anothervision.tokyo/contents/details/chikamichi_shimbun)**(2019)
- a take-home riddles kit（持ち帰り謎） that is packed in a set of one copy that resembles an actual newspaper
- created with some members of AnotherVision（東京大学謎解き制作集団AnotherVision）
- utilizes LINE Bot
- purchased by 4000+ people
- on sale in [booth.pm](https://anothervision.booth.pm/items/1472887) as of 2023
